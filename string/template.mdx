# string/template

`template :: Template -> Function -> String`

Creates a template function that returns a `String`.

## Example

```javascript
import { log, pipe, run, replace, $ } from 'mojiscript'

const state = { first: 'Luke', last: 'Skywalker' }

const nameTemplate = $`${'first'} ${'last'}`

const main = pipe ([
  nameTemplate,
  log
])

run ({ state, main })
// => "Luke Skywalker"
```

## Returns

Formatted string.
