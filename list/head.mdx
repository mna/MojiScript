# list/head

`head :: Iterable -> Any`

Takes an `Iterable` and returns the head of the `Iterable`.

## Example

```javascript
import { log, pipe, run, head } from 'mojiscript'

const main = pipe ([
  [ 1, 2, 3 ],
  head,
  log
])

run ({ main })
//=> 1
```

## Parameters

| Name | Type | Description |
| ---- | ---- | ----------- |
| iterable | `Iterable`  | `Iterable` to get the first value from. |

## Returns

Returns the head of `Iterable`
