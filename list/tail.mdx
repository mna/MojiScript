# list/tail

`tail :: Iterable -> Array`

Takes an `Iterable` and the tail of the `Iterable` as an `Array`.

## Example

```javascript
import { log, pipe, run, tail } from 'mojiscript'

const main = pipe ([
  [ 1, 2, 3 ],
  tail,
  log
])

run ({ main })
//=> [2, 3]
```

## Parameters

| Name | Type | Description |
| ---- | ---- | ----------- |
| iterable | `Iterable`  | `Iterable` to get the tail from. |

## Returns

Returns the tail of `Iterable`
