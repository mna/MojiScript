# list/filter

`filter :: Function -> Iterable -> Array`

Takes a predicate and an `Iterable` and returns an `Array` of the same type containing the members of the given filterable which satisfy the given predicate.

An `Iterable` could be an `Array`, an `Iterator`, or `async Iterator`.

## Array Example

```javascript
import { filter } from 'mojiscript'

const isEven = num => num % 2 === 0
const array = [ 1, 2, 3, 4, 5 ]

filter (isEven) (array) //=> [ 2, 4 ]
```

## Iterator Example

```javascript
import { filter, range } from 'mojiscript'

const isEven = num => num % 2 === 0

filter (isEven) (range (1) (6)) //=> [ 2, 4 ]
```

## Async Example

```javascript
import { filter, pipe, sleep } from 'mojiscript'

const isEven = num => num % 2 === 0
const asyncIsEven = pipe ([
  sleep (1000),
  isEven
])

const array = [ 1, 2, 3, 4, 5 ]

filter (asyncIsEven) (array) //=> Promise ([ 2, 4, 6 ])
```

## Parameters

| Name | Type | Description |
| ---- | ---- | ----------- |
| predicate | `Function`  | Function takes and input and returns a `Boolean` |
| iterable | `Iterable`  | `Iterable` to apply the predicate to. |

## Returns

Returns an `Array` filtered by the `predicate`.
